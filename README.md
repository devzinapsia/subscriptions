# Subscription Quotas Enhancement
Este módulo extiende la funcionalidad nativa de Suscripciones de Odoo para permitir el control de cuotas numeradas.

## Características:
- Definición de cantidad máxima de cuotas.
- Seguimiento de la próxima cuota a facturar.
- Leyenda dinámica en las líneas de factura: "Cuota X de Y".
